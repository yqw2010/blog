extend作为一个静态函数被调用，null被绑定，但是因为满足\只有一个”参数（boolean除外）的条件 ， 将函数的this环境（window）extend进去，return的对象就是window对象，所以得到的结果是

$.extend.apply( null, [ true, { “a” : 1, “b” : 2 } ].concat( { “c” : 3, “d” : 4 } ) );

$.extend.apply( $, [ true, { “a” : 1, “b” : 2 } ] );
